# cabinet-test

1.
```
npm install
```
2. Чтобы открылись контакты предварительно запускаем **json-server на localhost:3000** - это важно.
```
json-server --watch db.json
```
3. Запускаем проект
```
npm start
```
В этом проекте данные подтягиваются **через Fetch из файла db.json** имитирующий сервер.
